Comparativo de Linguagens
Tipos de Linguagens Uso Prós Contras Opnião
C++ Programas de computador.
Sistemas operacionais.
Desenvolvimento de aplicativos móveis.		 Altamente versátil.
Rápida e potente.
Permite maior controle.
Funciona bem entre diferentes dispositivos e plataformas.		 Pode ser complicada para iniciantes.
Sua estrutura orientada a objetos é menos eficiente 		É a linguagem mais indicada para aprendizagem apesar de sua sintexe complexa.
C# Aplicações web.
Aplicações móveis.		 Tempos de compilação e execução rápidos.
Grande variedade de frameworks.		 Aprendizado inicial mais difícil.
Não se traduz bem entre plataformas, pois é dependente do ambiente .NET. 		É uma linguagem requisitada no mercado devido ao seu ótimo desempenho e versatilidade
Java Desenvolvimento móvel.
Desenvolvimento de apps.
Aplicações desktop.		 Altamente versátil.
Simples de aprender, escrever, compilar e debugar.
Novos frameworks aceleram o desenvolvimento.		 Tempo de execução lento.
Consome relativamente bastante memória.		 é uma linguagem de sintexe simples e que é possível tudo.
JavaScript Desenvolvimento web front-end.
Desenvolvimento de jogos.
Aplicações web.		 Código aberto.
Funciona muito bem com outras linguagens de programação.		 Problemas de segurança.
Desempenho instável entre diferentes navegadores.		 É uma das linguagens mais utilizadas junto com o html e css para o desenvolvimento web Front-End.
PHP Desenvolvimento web.
Aplicações desktop.
Programas de computador.		 Código aberto.
Bem estabelecida para desenvolvimento web.		 Questões de segurança.
Desenvolvimento mais lento.
Menos ferramentas de debugging (resolução de problemas).		 É uma linguagem essencial na programação Back-End.
Phyton Desenvolvimento web back-end.
Aplicações desktop.
Ciência de dados.		 Código aberto.
Altamente escalável.		 Consome bastante memória.
Não é ideal para desenvolvimento de aplicações móveis.		 É uma linguagem muito popular entre os desenvolvedores devido a facilidade de sua sintexe.
